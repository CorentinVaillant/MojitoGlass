# Mojito Glass 🥤

This is a utility library for myself and friends.

## Build 

You can build with cmake.
Usage as an external library has **not** been tested yet; this will be tested later during development.

You can build and test with :

```bash
cmake -B build -DBUILD_TESTING=ON
cmake --build build
```

You will find a test executable inside the `bin` folder.
