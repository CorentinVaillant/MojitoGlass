# Mojito Glass 🥤

This is a util library, for myself and friends.

## Build 

You can build with cmake.
Usage as a external library has **not** been test yet, it will be further in developpement.

You can build and test with :

```bash
cmake -B build -DBUILD_TESTING=ON
cmake --build build
```

You will find a test executable inside the `bin` folder.
